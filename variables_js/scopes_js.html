Types of Scopes:

  • Global Scope
  • Function/local Scope
  • Block Scope

Global Scope:

An Variable which is declared outside the function which can also used inside the function is known as global Scope.

Example:
var name = "yadeshwar";
function global_scope() {
console.log(name); // output: yadeshwar
}
global_scope();

Function/Local Scope:

An Variable which declared inside the function is cant used in outside the function this is called function/local Scope.
Example:
function local_Scope(){
var name = "yadeshwar";
console.log(name);
}
local_scope();
console.log(name); // throws error

Block Scope:

An variable which is declared in a function within an block it can be used in the same block and it cant be used in other blocks in the function this is called block Scope. In block scope variables should declare by let or const.
Example:
function block_scope() {
var name = "yadeshwar"; // function scope

let age = 20; // block code
const gender = "male";
}
}
block_scope();
